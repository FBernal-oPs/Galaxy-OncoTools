Welcome to OncoTools!


OncoTools is a Galaxy instance that will host dedicated tools intended for interpretation of data from high-throughput screens in cancer biology.

List of tools currently available:

    oncoEnrichR - Prioritization and functional interrogation of human genesets in the context of cancer. The tool is intended for exploratory analysis and prioritization of a gene list from high-throughput cancer biology experiments. The tool queries a number of high-quality data resources in order to assemble useful gene annotations and analyses in an interactive report. The tool is also available as an R package (see GitHub page).

OncoTools is supported by the Centre for Cancer Cell Reprogramming (CanCell) at the University of Oslo, with key contributions by Elixir (Oslo node).
